Some free resources for learning about machine learning

    Books

  • A public domain book for a machine learning course by Michael S. Gashler, University of Arkansas
  • A book for a machine learning/data mining course by Aaron Hertzmann and David Fleet, University of Toronto
  • A book on statistical learning by Trevor Hastie, Robert Tibshirani, and Jerome Friedman.

    • Web tutorials

  • Lecture notes from a course on machine learning at Stanford
  • Tutorials on deep learning

    • Video lectures

  • Andrew Ng's video lectures on machine learning
  • Pedro Domingos' video lectures on machine learning
  • Daphne Koller's video lectures on probabilistic graphical models
  • Bill Howe's video lectures on data science
  • Nando de Freitas video lectures on machine learning
  • Hugo Larochelle's video lectures on neural networks and machine learning topics
  • Yaser Abu-Mostafa's machine learning and neural network lectures

    • Other lists of resources

  • Eren Golge's Blog
  • Metacademy's list of machine learning topics

If any of these resources ever ask you to create an account, watch ads, take a survey, register, etc., please let us know, so we can remove it. We appreciate suggestions about other free (in the strict sense) learning resources.
