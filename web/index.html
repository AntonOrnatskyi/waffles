Waffles is a toolkit of machine learning algorithms. These tools:
  • Use a command-line interface,
  • Compile and run across platforms,
  • Are licensed under the LGPL (or a more permissive license),
  • Are thin wrappers around functionality in a well-documented C++ class library.

Waffles might be described as "Machine Learning meets the Unix philosophy". That is, we provide a huge collection of simple command-line tools that you can chain together to do whatever you want, and to do it your own way. These tools are designed to be highly flexible, and to provide functionality across a broad spectrum of operations related to machine learning.

Waffles is especially rich in unsupervised techniques, such as non-linear dimensionality reduction, but also provides numerous supervised learning algorithms, collaborative filtering algorithms, data transformations, linear-algebraic operations, optimization methods, data generation tools, visualization tools, etc.

There is no need to memorize any commands. Waffles contains a graphical wizard tool that will guide you through forms to generate a command to will do exactly what you want...

...but you will never get "locked-in" to this GUI, because it prints out the command that will perform the desired task. You can paste this command into a shell to do the job immediately, or you can paste it into a script that will automate your processes. When you're ready to turn your scripted process into a high-performance application, all of the functionality is also available through our class library. Your apps need not be chained to any virtual machine.

To see if Waffles provides the functionality you need, just take a look at the documentation.

Matrix operations are also supported. For example, let's compute C=ATB†.

Command:
waffles_transform transpose a.arff > a_trans.arff
waffles_transform pseudoinverse b.arff > b_inv.arff
waffles_transform multiply a_trans.arff b_inv.arff > c.arff
Let's cluster some data. First, we'll use a nominaltocat filter to convert it to continuous values, then we'll cluster with three different algorithms. We'll use 3 clusters in each case.

Command:
waffles_transform nominaltocat mydata.arff > real.arff
waffles_transform agglomerative real.arff 3 > agglomerative.arff
waffles_transform kmeans real.arff 3 > kmeans.arff
waffles_transform kmedoids real.arff 3 > kmedoids.arff
