A collection of command-line tools for researchers in machine learning, data mining, and related fields. All of the functionality is also provided in a clean C++ class library. Demo apps are included to show how to use the class library.
by: Mike Gashler

Builds on 32 and 64-bit platforms, including Linux, Windows, OSX, etc.
Distributed under the LGPL license.

Waffles is a toolkit of machine learning algorithms. These tools:
  • Use a command-line interface,
  • Compile and run across platforms,
  • Are licensed under the LGPL (or a more permissive license),
  • Are thin wrappers around functionality in a well-documented C++ class library.

Waffles might be described as "Machine Learning meets the Unix philosophy". That is, we provide our functionality in the form of simple command-line tools. This is preferable to a GUI because:

  • You are free to use tools from other toolkits along with our tools.
  • You are free to use your own custom tools along with our tools.
  • You are free to write scripts that automate your experiments.
  • You are free to use our tools in ways that we could not foresee.
  • etc.

Waffles is especially rich in unsupervised techniques, such as non-linear dimensionality reduction, but also provides numerous supervised learning algorithms, collaborative filtering algorithms, data transformations, linear-algebraic operations, optimization methods, data generation tools, visualization tools, etc.

What if you don't like memorizing commands? Waffles contains a graphical wizard tool that will guide you through forms to generate the command that will do exactly what you want...

...but you will never get "locked-in" to this GUI, because it prints out the command that will perform the desired task. You can paste this command into a shell to do the job immediately, or you can paste it into a script that will automate your processes.

All of our command-line tools are thin wrappers around functionality in our open source C++ class library. So, when you're ready to turn your scripted process into a high-performance application, you can link to our libary, and you're ready-to-go. Your apps need not be chained to a virtual machine that limits runtime performance and encumbers the distribution of your software.

If you want to know whether Waffles provides the functionality you need, the next step is to take a look at the documentation.
